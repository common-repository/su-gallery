== Description ==
 
Su-Gallery Plugin is used to create multiple photo galleries with fancybox 3.  Integrates with your website in less than 20 seconds. use shortcode "[su_gallery]". 

== Installation ==
 
1. Download su-gallery.zip and extract it from wp-content\plugins (plugin folder).
1. Activate Su-Gallery plugin and use shortcode [su_gallery].
